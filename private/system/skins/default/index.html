Stuffed System is an advanced framework for web based applications written in Perl. It uses both object oriented and functional code, depending on the situation, and doesn't limit developers to only one of these approaches.

The fact that you see this message means that this particular web application is making use of the great Stuffed System features, including skins support, templates compilation for faster processing, plugins, built-in internationalization, full mod_perl compatibility. If you are interested in using Stuffed System in your own applications please contact Stuffed Guys for further details.

This version of Stuffed System is fully optimized to work with utf8 encoding in Perl.
